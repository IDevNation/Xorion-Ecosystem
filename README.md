# Xorion Ecosystem

Xorion Ecosystem is the unified repository for the Xorion blockchain network, wallet infrastructure, developer SDK, governance modules, runtime services, and user-facing applications.

## Structure

- `core/` - blockchain and core network components
- `node/` - node layer
- `sdk/` - wallet SDK and shared developer logic
- `apps/` - wallet extension, dashboard, GUI
- `modules/` - governance, storage, privacy, runtime
- `website/` - public-facing website
- `docs/` - technical and user documentation

## Vision

Xorion is building a modular Web3 ecosystem that connects:
- blockchain infrastructure
- wallet products
- explorer and scan tooling
- governance
- privacy and storage modules
- developer tooling

## Current Components

- Xorion blockchain
